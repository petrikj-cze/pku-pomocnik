## PKU Sledování jídelníčku

Tento projekt má za cíl pomoci pacientům s diagnózou PKU (Nízkobílkovinové diety) sledovat jejich denní jídelníček.

# Hlavní funkce
	- Přidání a odebrání libovolných suroviny
	- Spojování surovin do hotových jídel
	- Výběr skutečně použitých surovin u hotových jídel
	- Graficky přehledný ukazatel pHe v aktuálním dni
